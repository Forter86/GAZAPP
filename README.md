# Gazprom App

Telegram Mini App для трудоустройства в Газпром трансгаз Сургут.

## Установка: тест пуша 

```bash
npm install
```

## Запуск

```bash
# Фронтенд
npm run dev

# Бэкенд
npm run server

# Оба одновременно
npm run dev:all
```

## Деплой

Фронтенд деплоится на Vercel, бэкенд на Railway.
